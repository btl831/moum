# MOUM

***BTL831팀의 두번째 프로젝트***

BTL831의 첫프로젝트 보완 + 새로운 MOUM 추가
------

목적: 사실상 첫번째 크롤링과 데이터를 넣는것을 제외하면 프론트엔드에 비중이 컸다. 그래서 백엔드와 웹게임을 추가하여 노래MOUM이 아닌 다양한 분야**MOUM**으로 변화 시킬려고 한다.

목표: 백엔드 구현 및 페이지 배포,이슈탭 사용

매주 화요일 7pm 회의 및 발표 스터디 진행

참여자: 신봉규,박찬현
